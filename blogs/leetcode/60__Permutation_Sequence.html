leetcode 6. Permutation Sequence



https://leetcode.com/problems/permutation-sequence/

problem: 
The set [1,2,3,...,n] contains a total of n! unique permutations.

By listing and labeling all of the permutations in order, we get the following sequence for n = 3:

"123"
"132"
"213"
"231"
"312"
"321"
Given n and k, return the kth permutation sequence.

Note:

Given n will be between 1 and 9 inclusive.
Given k will be between 1 and n! inclusive.
Example 1:

Input: n = 3, k = 3
Output: "213"
Example 2:

Input: n = 4, k = 9
Output: "2314"

我的思路：
利用递归，每次计算出最开始的一个数字 

class Solution(object):
    def getPermutation(self, n, k):
        """
        :type n: int
        :type k: int
        :rtype: str
        """
        thelist = [i for i in range(1, n+1, 1)]
        return self.permutating(thelist, k)
    
    def permutating(self, thelist, k):
        if k == 1:
            return ''.join([str(x) for x in thelist])
        tot = 1
        for i in range(1, len(thelist)):
            tot *= i
        n = int((k - 1) / tot)
        pre = str(thelist[n])
        del thelist[n]
        return pre + self.permutating(thelist, k - n * tot)
