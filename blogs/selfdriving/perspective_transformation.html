透视转换 perspective transformation



由于透视的关系，一个平面上方方正正的图形在照片上看上去会显示处一个梯形的形状，这就是物体在照片上的透视投影。
我们需要把这个透视的形状还原成原始的形状。

OpenCV 提供了相关的函数来帮助我们做这个事情。

import cv2
import numpy as np
import matplotlib.pyplot as plt
import matplotlib.image as mpimg

img = cv2.imread('test_image2.png')

src = np.float32([[100, 100], [95, 200], [220, 250], [200, 70]])
dst = np.float32([[100, 100], [100, 200], [200, 200], [200, 100]])
M = cv2.getPerspectiveTransform(src, dst)
warped = cv2.warpPerspective(img, M, (img.shape[1], img.shape[0]), flags=cv2.INTER_LINEAR)


这样 我们得到了 perspective transform 之后的 warped 图像。

当然你可以预先显示一下这些点，观察一下看看是不是设置的很恰当。
