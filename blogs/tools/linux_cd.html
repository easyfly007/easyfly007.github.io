利用 alias 提高 linux cd 命令使用效率



linux 环境各中最常用的是cd 命令，然而我们仍然有一些办法来提高你的效率。

1. 对于你常用的工作路径，利用 alias 直接 cd 到对应位置。 

alias cdw cd /YOUR/WORK/DIR


2. 对于返回到上层目录以及上上层目录 
alias . cd ../
alias .. cd ../../
alias ... cd ../../../
alias .... cd ../../../../


3. 很多时候我们访问了 cd 命令之后往往会调用 ls，那么可以这样做来省去这个 ls： 
  
cdls() {
    cd "${1}";
    ls;
}
alias cd='cdls'
4. 在特定后缀的文件名中检索 
  alias cha 'find . name "*" -print | xargs grep -n -i'
  alias chacc 'find . name "*.cc" -print | xargs grep -n -i'
